# 📊 AI Resume Analytics Platform

An AI-powered resume screening and candidate ranking system that evaluates resumes against job descriptions using NLP techniques and provides intelligent hiring insights.

---

## 🚀 Live Demo
👉 https://ai-resume-analytics-platform-ngxljxngt744zcgvywzntd.streamlit.app/

---

## 🧠 Project Overview

This project simulates a real-world recruitment analytics system used for:

- Matching resumes with job descriptions
- Extracting relevant skills
- Ranking multiple candidates
- Providing ATS-style scoring
- Generating structured hiring reports

---

## ⚙️ Features

- 📄 Multi-candidate resume analysis
- 🧠 TF-IDF based text vectorization
- 📊 Cosine similarity scoring for job match
- 🎯 ATS-style scoring system
- 🔍 Skill gap detection (matched vs missing skills)
- 🏆 Automatic candidate ranking
- 📥 Downloadable JSON analytics report
- 📈 Visual skill comparison charts

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- NLP (TF-IDF Vectorization)  
- Cosine Similarity  
- Matplotlib  

---

## 📌 How It Works

1. User inputs job description  
2. Multiple resumes are entered  
3. Text is cleaned and processed  
4. TF-IDF converts text into numerical vectors  
5. Cosine similarity calculates match score  
6. Candidates are ranked based on relevance  
7. ATS score + insights are generated  

---

## 🏆 Output

- Ranked candidate list  
- Match percentage  
- Skill gap analysis  
- Visual charts  
- JSON report download
